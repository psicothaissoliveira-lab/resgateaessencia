# PRD - O Retorno à Essência

## Problem Statement Original
Recriar o site "O Retorno à Essência" (https://oresgateaessencia.mgx.world/) alterando o valor de R$ 197 para R$ 47.

## Arquitetura
- **Frontend**: React.js com Tailwind CSS
- **Design**: Dark mode elegante com tema "Midnight Garden" (Deep Wine/Stone + Rose Gold)
- **Tipografia**: Playfair Display (headings) + Manrope (body)

## User Personas
- Mulheres que buscam cura emocional e reconexão
- Vítimas de relacionamentos tóxicos ou manipulação emocional
- Mulheres que perderam sua identidade em relações abusivas

## Core Requirements (Static)
- Landing page single-page com múltiplas seções
- Hero section impactante
- Descrição da jornada de 5 aulas
- Seção de preço com valor R$ 47
- CTAs de conversão
- Design feminino e emocional
